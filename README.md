# 🎮 CoinFlip 翻金币小游戏
基于 Qt 6 开发的经典翻金币益智小游戏，支持 macOS / Windows 双平台。

## 📖 项目介绍
- 经典玩法：点击金币翻转自身及上下左右相邻金币，目标是将所有金币翻为正面
- 多关卡系统：内置多难度关卡，逐步挑战
- 完整音效：点击、返回、胜利音效全覆盖
- 流畅动画：金币翻转动画、胜利弹窗弹跳动画
- 跨平台支持：一套代码，macOS / Windows 均可编译运行

## 🛠️ 开发环境
- Qt 6.10.2
- CMake 构建系统
- C++ 17

- 新加入了 ##无线模式 你可以一直尝试挑战不同的关卡了!
- 新加入了 ##重新开始 你可以重新挑战关卡了!
